Test.Plutus.Validator

Description

A helper module mostly wrapping the Alonzo.Tools' evaluateTransactionExecutionUnits with a much simpler API (just a plutus script).

This is generally handy to measure the execution of Plutus code outside of any context (e.g. an implementation of a data-structure on-chain or, as here, data encoders).
